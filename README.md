# Vermilion Voice

Obsidian 本地语音识别插件 — ONNX + WebGPU，无需云端。

[English](./docs/README.md) | [简体中文](./docs/README_zh.md) | [架构设计](./docs/02-architecture.md)

## 快速开始

```bash
npm install
npm run build     # 生产构建 → plugin/ + vermilion-voice-x.x.x.zip
npm run dev       # 开发构建（含 sourcemap）
```

构建产物在 `plugin/`，zip 包用于 [GitHub Release](https://github.com/sujinlin00/vermilion-voice/releases)。

## 安装使用

从 Release 下载 `vermilion-voice-x.x.x.zip`，解压到 `.obsidian/plugins/vermilion-voice/`。

详见 [docs/README.md](./docs/README.md)。
